Что такое TypeScript

Что такое TypeScript:

  • Надстройка над JavaScript
  • Обратно совместим с JavaScript
  • Компилируется в JavaScript
  • Позволяет явно указывать статические типы

Чем полезен TypeScript

Чем полезен TypeScript:

  • Убирает duck-typing
  • Позволяет использовать формальные контракты
  • Позволяет IDE лучше вам помогать
  • Обнаруживает ошибки на этапе компиляции
